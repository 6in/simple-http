# http アプリケーション

## 概要

静的なHttpサーバーを起動します。

## コマンドラインオプション

```
Usage:
  http --port=<port> [--root=<root>]
  http (-h | --help)
  http --version

Options:
  --port=<port> port no .
  --root=<root> document root directory [default: .].
  -h --help     Show this screen.
  --version     Show version.
```
